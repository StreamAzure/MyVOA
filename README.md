# 自用的VOA听力练习APP

走路的时候想练听力，有时候一句话需要反复倒退听，但是得拿起手机看准“上一句”的按钮在哪里才能点到，还容易不小心瞥到原文被剧透。

现在只需要把手机握在手里用拇指划划就可以了~不耽误看路

- 沉浸式播放界面
- 支持左滑/右滑手势切换上一句/下一句
- 支持双击屏幕暂停/继续播放
